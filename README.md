# 🧠 FocusFlow AI — Smart Pomodoro for Deep Work

FocusFlow is a simple and minimal Pomodoro-style web app that I built to improve my own focus during study and coding sessions. The idea is to keep things clean, distraction-free, and actually useful — not overloaded with features.

## 🚀 What it does

* ⏱️ Lets you set custom focus sessions (like 25, 50, or 90 minutes)
* 📊 Tracks how much time you actually focus and shows simple stats
* 🧠 Gives small insights based on your past sessions
* 🚫 Lets you log distractions during a session
* 🌙 Clean, minimal UI inspired by Notion
* 🧘 Fullscreen “Deep Focus Mode” to remove distractions
* 🔔 Keeps track of your timer even if you switch tabs

## 🛠️ Tech Stack

* React + Tailwind CSS for UI
* Chart.js / Recharts for visualizing data
* LocalStorage to save sessions (no login needed)
